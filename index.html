Angular DRY Demo

DRY demos?

Your code is DRY, why aren't your demos?

Why?

Live demos with code can be a pain. You know what I'm talking about, &lt; and &gt; running around everywhere, text hugging the left side of the page, and on top of all that you're just repeating the code from the actual demo anyway.

Don't get me started on what happens when you later make changes to those demos... gross.

So who you gonna call?

Angular DRY Demo! It's is a set of directives that address the pain points of writing live demos with code. It takes a snapshot of the actual code running in your demo, then writes it back out wherever you like in a format fit for showcasing.
